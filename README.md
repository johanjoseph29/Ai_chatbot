1. Clone the repository:

2. Install Dependencies:
   ```bash
   flutter pub get
   ```
3. Run the app:
   ```bash

   ```

### Backend Setup

1. Navigate to the backend folder:
   ```bash
   cd backend
   ```
2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
   
**requirements.txt:**
```plaintext
Flask==2.3.3
Flask-Cors==3.0.10
google-generativeai==0.2.2
textblob==0.17.1
```

3. Run Flask API:
   ```bash
   python app.py
   ```
   API URL: `http://127.0.0.1:5000/chat`
